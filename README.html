04 / Technology

Three Technologies. Each Best-in-Class.

🌐

Bright Data Scraping Browser

puppeteer-core · WS connect

Instead of a local Chromium, NemoForge dials into Bright Data's residential browser cluster over WebSockets. The agent runs inside a real browser on a real residential IP.

  • TLS fingerprint matches real Chrome
  • Residential IP from 72M+ pool
  • Built-in CAPTCHA resolution
  • Web Unlocker for API-level bypass
🖱️

Ghost Cursor

ghost-cursor · bezier paths

Most bots move the mouse in straight lines. Ghost Cursor generates statistically human Bézier curves — with overshoot, fatigue, and natural deceleration.

  • Bézier-curve trajectories per move
  • Injected visible cursor for demo viewport
  • Random clicks defeat passive CAPTCHAs
  • Scroll entropy mimics reading behaviour
🤖

NVIDIA Llama-3.1-8b-instruct

NVIDIA NIM API · /v1/chat

The cleaned DOM text (up to 15k chars) is sent to meta/llama-3.1-8b-instruct via the NVIDIA NIM API. The model acts as a human analyst reading the page.

  • No CSS selectors — schema-free extraction
  • Natural language prompt → structured answer
  • Markdown-formatted analyst report
  • Works even if site restructures completely
